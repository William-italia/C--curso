# ExercicioSinal
Leia um número inteiro e exiba "Negativo", "Zero" ou "Positivo", conforme o caso.

*Como Executar:*
```
Dotnet run
```
## Exemplos
```
--- Vou dizer se o numero digitado é zero ou negativo/positivo ---

Digite um número: 70
Positivo
``` 
Ou
```
Digite um número: -70
Negativo
```
Ou
```
Digite um número: 0
Zero
```
